F-Droid - это каталог свободных приложений для Android. С помощью клиентского приложения F-Droid можно легко находить, устанавливать и отслеживать обновления на вашем устройстве.

Приложение подключается к любым репозиториям, совместимым с F-Droid. По умолчанию используется репозиторий f-droid.org, который содержит только подлинно свободное программное обеспечение.

Сам Android является открытым в том смысле, что вы можете свободно устанавливать APK-файлы откуда угодно, но есть много веских причин использования F-Droid в качестве менеджера свободных приложений:

* Получение уведомлений о появлении обновлений
* Автоматическая загрузка и установка обновлений
* Отслеживание устаревших приложений и бета-версий
* Фильтрация несовместимых с устройством приложений
* Поиск приложений по категориям и описаниям
* Доступ к ассоциированным URL для пожертвований, исходного кода и т.д.
* Обеспечение безопасности путем проверки подписей индекса репозиториев и хэшей APK
